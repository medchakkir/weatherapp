## üìã Project Overview

Weather App is a clean, user-friendly web application that provides real-time weather information for any city worldwide. It features a beautiful, dynamic UI with smooth animations, error handling, and geolocation support. The app uses the OpenWeatherMap API to fetch current weather data and displays it in an intuitive, visually appealing interface.

## ‚ú® Features

- **City Search**: Search for weather by city name with real-time results
- **Geolocation Support**: Get weather for your current location with one click
- **Real-time Weather Data**: Current temperature, weather conditions, and detailed metrics
- **Loading States**: Visual spinner indicator during API calls
- **Error Handling**: User-friendly error messages for various scenarios
- **Empty State**: Helpful message when no city is searched yet
- **Debounced Search**: Reduces API calls with intelligent input debouncing
- **Smooth Animations**: Fade-in and slide-up transitions for weather data updates
- **Dynamic Backgrounds**: Background changes based on time of day and weather conditions
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Accessibility**: ARIA labels and semantic HTML for screen readers

## üîß API Configuration

This app uses the OpenWeatherMap Current Weather Data API:

- **Endpoint**: `https://api.openweathermap.org/data/2.5/weather`
- **Free Tier**: 60 calls/minute, 1,000,000 calls/month
- **Documentation**: [OpenWeatherMap API Docs](https://openweathermap.org/current)
